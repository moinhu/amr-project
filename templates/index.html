📘 Project Overview

This project focuses on predicting antimicrobial resistance (AMR) using machine learning techniques. The system analyzes genomic and biological features to classify whether a microorganism is resistant or susceptible to antibiotics. Multiple models such as Random Forest, Gradient Boosting, SVM, and Logistic Regression are used and evaluated using accuracy, AUC, and cross-validation.
